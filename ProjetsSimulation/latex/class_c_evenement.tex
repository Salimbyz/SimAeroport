\doxysubsection{Detailed Description}
Classe de base pour la gestion des événements dans un aéroport. 

Cette classe représente un événement général dans un aéroport. Elle est conçue pour être héritée par des classes spécifiques à chaque type d\textquotesingle{}événement (atterrissage, décollage, etc.). Elle gère le temps de début de l\textquotesingle{}événement.
