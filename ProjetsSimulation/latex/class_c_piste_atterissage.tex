\doxysubsection{Detailed Description}
Classe représentant une piste d\textquotesingle{}atterrissage. 

Cette classe gère les opérations liées à une piste d\textquotesingle{}atterrissage, y compris la gestion des avions en attente d\textquotesingle{}atterrissage.
