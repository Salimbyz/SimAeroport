\doxysubsection{Detailed Description}
Classe représentant une ressource générale (pouvant être une piste, une porte, etc.). 

Cette classe sert de base pour différentes ressources dans un aéroport, telles que les pistes d\textquotesingle{}atterrissage, de décollage ou les portes d\textquotesingle{}embarquement.
